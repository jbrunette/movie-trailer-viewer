Movie Trailer Viewer

INSTALLATION
=========================
1. This application requires Python 2.7 or higher.  Get python from: https://www.python.org/downloads/
2. Clone the github repository at: https://github.com/jbrunette/movie-trailer-viewer
3. Run entertainment_center.py.  This should open a web browser with the movie trailer page.
4. Enjoy the movie trailers.
